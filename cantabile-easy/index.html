용어 사전
Cantabile
VST 플러그인을 실시간으로 돌리는 프로그램. 음악인 + 스트리머 표준 도구.
ASIO
소리 지연(레이턴시)을 줄이는 드라이버. 오디오 인터페이스에 필수.
VST Plugin
Cantabile 안에 꽂는 가상 악기 / 이펙터. Reverb, EQ 등.
Reverb
공간 잔향 효과. 노래방 느낌 → 홀 느낌까지 조절 가능.
EQ (이퀄라이저)
소리 주파수를 올리거나 내림. 목소리를 더 선명하게 만들어 줌.
Compressor
큰 소리는 줄이고, 작은 소리는 키워서 음량을 일정하게 맞춤.
Latency (지연)
마이크에서 소리가 나온 뒤 귀에 들릴 때까지의 시간차. 작을수록 좋음.
Audio Interface
마이크를 컴퓨터에 연결하는 장치. USB로 연결함.
OBS
무료 방송 / 녹화 프로그램. Cantabile 소리를 OBS로 전달해서 방송에 씀.
Buffer Size
한 번에 처리하는 소리 양. 작을수록 지연 줄지만 CPU 부하 올라감.
Section 01

Cantabile 가 뭐야? What is Cantabile?

노래할 때 마이크 소리에 리버브나 EQ 같은 효과를 실시간으로 입혀주는 프로그램이야. 방송, 녹음, 연습 모두 쓸 수 있어. Cantabile is a program that applies effects like reverb and EQ to your mic in real time. Perfect for streaming, recording, and practice.

+-- 어떤 상황에 쓰나? ---------------------------+ | | | 노래 연습 → 혼자 연습할 때 리버브 걸기 | | 방송 → OBS 에 이쁜 목소리 내보내기 | | 녹음 → DAW 없이 간단 녹음 | | | +-----------------------------------------------+
간단 비유: Cantabile = 노래방 기계의 컴퓨터 버전. 마이크를 꽂으면 실시간으로 이펙트 적용! Cantabile = karaoke machine for your PC. Plug in mic, get instant effects!
Section 02

내가 필요한 것 (장비 + 소프트웨어) What You Need (Gear + Software)

딱 2가지만 있으면 돼. 마이크 + 오디오 인터페이스. 이미 있으면 바로 설치로 넘어가! You only need 2 things: a microphone + an audio interface. Already have them? Jump to install!

Section 03

다운로드 / 설치 (5 단계) Download / Install (5 Steps)

총 5단계야. 천천히 따라가 보자! 5 steps total. Follow along!

  1. 1
    Cantabile 공식 사이트 접속Go to official site
    → cantabilesoftware.com 접속 → Visit cantabilesoftware.com
  2. 2
    "Download" 클릭 → Cantabile Lite 선택 (무료!)Click "Download" → Choose Cantabile Lite (Free!)
    Solo / Performer 는 유료야. Lite 로 시작하면 충분해. Solo / Performer are paid. Lite is enough to start.
  3. 3
    설치 파일 실행 → "Next" 계속 클릭Run installer → Keep clicking "Next"
    기본 설정 그대로 OK. 설치 경로 바꾸지 않아도 됨. Default settings are fine. No need to change install path.
  4. 4
    ASIO 드라이버 설치Install ASIO driver
    Focusrite 인터페이스 있으면 → Focusrite Control 설치
    USB 마이크만 있으면 → ASIO4ALL 설치 (무료, 구글 검색)     Focusrite interface → install Focusrite Control
    USB mic only → install ASIO4ALL (free, google it)
  5. 5
    마이크 / 인터페이스 연결 후 Cantabile 실행!Connect mic / interface, launch Cantabile!
    바탕화면 아이콘 더블클릭하면 시작. Double-click the desktop icon to start.
설치 체크리스트: [✓] cantabilesoftware.com 접속 [✓] Cantabile Lite 다운로드 [✓] 설치 완료 [✓] ASIO 드라이버 설치 [✓] 마이크 연결 + 실행 ← 여기까지 하면 됨!
Section 04

첫 설정 (Input / Output) First Setup (Input / Output)

Cantabile 처음 켰을 때 소리가 안 나오면 여기 설정부터 해! If no sound when you first open Cantabile, start here!

  1. 1
    상단 메뉴 → Tools → OptionsTop menu → Tools → Options
    단축키: Ctrl + ,Shortcut: Ctrl + ,
  2. 2
    Audio Engine 탭 선택Select Audio Engine tab
    여기서 드라이버 선택함.This is where you pick the driver.
  3. 3
    Driver Mode → ASIO 선택Driver Mode → Select ASIO
    드롭다운에서 ASIO 로 바꿔. (WASAPI 는 지연이 좀 있어.)Change dropdown to ASIO. (WASAPI has more latency.)
  4. 4
    Audio Device → 네 인터페이스 / 마이크 선택Audio Device → Select your interface / mic
    예: "Focusrite USB ASIO" 또는 "ASIO4ALL v2"e.g. "Focusrite USB ASIO" or "ASIO4ALL v2"
  5. 5
    Buffer Size → 256 또는 512 추천Buffer Size → Try 256 or 512
    숫자 작을수록 지연 줄지만 컴퓨터 힘들어해. 소리 끊기면 올려봐.Smaller = less delay but more CPU. Increase if audio glitches.
  6. 6
    OK 클릭 → 마이크 말해보기 → 소리 나오면 성공!Click OK → Talk into mic → If you hear sound, you're done!
Tools → Options → Audio Engine Driver Mode : ASIO Audio Device : Focusrite USB ASIO (또는 ASIO4ALL v2) Buffer Size : 256 (작으면 128, 느리면 512) Sample Rate : 44100 Hz → OK 클릭 후 마이크 테스트!

Section 06

녹음 / 방송 시작 (OBS 연결) Recording / Stream Start (OBS Link)

Cantabile 소리를 OBS 로 보내면 방송에서 이쁜 목소리 나와. VB-Cable 이라는 무료 프로그램 하나만 더 설치하면 돼. Send Cantabile audio to OBS for beautiful voice on stream. Just install one free program: VB-Cable.

Cantabile → OBS 연결 흐름: 마이크 → Cantabile (플러그인 체인 적용) → VB-Cable (가상 케이블) → OBS 오디오 소스 = "CABLE Output" → 방송 / 녹화 출력 ♪
  1. 1
    VB-Cable 설치Install VB-Cable
    → vb-audio.com/Cable (무료) → vb-audio.com/Cable (free)
  2. 2
    Cantabile → Output Port → "CABLE Input (VB-Audio)" 선택Cantabile → Output Port → select "CABLE Input (VB-Audio)"
  3. 3
    OBS → 오디오 소스 추가 → "오디오 입력 캡처" → "CABLE Output" 선택OBS → Add audio source → "Audio Input Capture" → select "CABLE Output"
  4. 4
    OBS 오디오 믹서에서 소리 확인 → 방송 시작!Check audio in OBS mixer → Start streaming!
TIP: 혼자 모니터링 하고 싶으면 헤드폰을 Cantabile Output 또는 인터페이스의 Direct Monitor 에 연결해! To monitor yourself, connect headphones to Cantabile output or your interface's Direct Monitor.
Section 07

자주 묻는 것 (FAQ) Frequently Asked Questions

소리가 아예 안 나와요! No sound at all! →
Tools → Options → Audio Engine 에서 ASIO 드라이버 선택됐는지 확인. 마이크/인터페이스가 USB에 제대로 연결됐는지 확인. Check Tools → Options → Audio Engine, make sure ASIO driver is selected. Verify mic/interface is properly connected via USB.
소리에 잡음(윙윙)이 있어요. There's hum/noise in the sound. →
USB 허브 말고 컴퓨터 본체에 직접 꽂어봐. MEqualizer 로 60-120Hz 구간을 살짝 내려도 돼. Try plugging directly into your PC instead of a USB hub. Also try cutting 60-120Hz with MEqualizer.
소리가 끊겨요 (딱딱 소리). Audio is glitching (crackling sounds). →
Buffer Size 를 키워봐 (128 → 256 → 512). CPU 과부하일 수 있어. Try increasing Buffer Size (128 → 256 → 512). Might be CPU overload.
플러그인이 목록에 안 나와요. Plugin doesn't appear in the list. →
Tools → Options → Plugin Folders 에서 VST 폴더 경로 추가 후 "Scan" 클릭. 64bit 플러그인인지 확인 (32bit 호환 안 될 수 있음). Add VST folder path in Tools → Options → Plugin Folders then click "Scan". Make sure plugin is 64-bit (32-bit may not work).
OBS 에서 Cantabile 소리가 안 들려요. OBS can't hear Cantabile audio. →
VB-Cable 설치됐는지 확인. Cantabile Output = "CABLE Input", OBS 소스 = "CABLE Output" 으로 설정됐는지 확인. Check VB-Cable is installed. Cantabile Output = "CABLE Input", OBS source = "CABLE Output".
Cantabile 유료 버전이 필요한가요? Do I need the paid version? →
처음엔 Lite (무료) 로도 충분해! 나중에 더 많은 플러그인 동시 사용이나 세트리스트 기능 쓰고 싶을 때 업그레이드 하면 돼. Lite (free) is plenty for starting out! Upgrade later if you need more simultaneous plugins or setlist features.
