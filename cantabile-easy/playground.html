Playground — 직접 해보자 Playground — Try It Yourself

슬라이더 움직이면서 소리 원리 익히기 · 시뮬레이션 only Move sliders to learn how audio works · Simulation only
Mini 도구
섹션
1. 버퍼 사이즈 1. Buffer Size 2. 리버브 다이얼 2. Reverb Dial 3. EQ 밴드 3. EQ Bands 4. 컴프레서 4. Compressor 5. 마이크 볼륨 5. Mic Volume 6. 플러그인 체인 6. Plugin Chain
용어
Buffer Size
작을수록 지연 줄지만 CPU 부하 증가.
Reverb
공간 잔향. Dry=원음 / Wet=이펙트.
EQ
주파수 대역별 음량 조절.
Compressor
Ratio 높을수록 음량 압축 강함.
Plugin Chain
플러그인 순서가 소리에 영향 미침.
Widget 01

버퍼 사이즈 — 지연 시간 시뮬 Buffer Size — Latency Simulation

버퍼 사이즈를 줄이면 지연이 줄어. 하지만 너무 줄이면 소리가 끊겨! Smaller buffer size = less latency. But too small = audio glitches!

Buffer Size 256
Widget 02

리버브 다이얼 — Soft / Large Reverb Dial — Soft / Large

다이얼을 돌려서 공간감 변화를 보자. Dry = 원음, Wet = 이펙트 양. Turn the dial to see space change. Dry = original, Wet = effect amount.

Room
Wet 양 Wet Amount 30%
Widget 03

EQ 밴드 — 주파수 시각화 EQ Bands — Frequency Visualization

각 주파수 대역을 올리거나 내려봐. 목소리에서 뭘 잘라낼지 느껴보자. Boost or cut each frequency band. Feel what to cut from your voice.

Widget 04

컴프레서 — Ratio 변경 Compressor — Change Ratio

Ratio 가 높을수록 큰 소리를 더 많이 줄여. 일정한 볼륨 유지에 필요해. Higher ratio = more compression of loud sounds. Keeps volume steady.

Threshold -12 dB
Ratio 4:1
Widget 05

마이크 볼륨 — 입력 시뮬 Mic Volume — Input Simulation

슬라이더로 마이크 입력 게인을 조절해봐. 클리핑(빨간 영역)이 생기면 소리가 깨져! Adjust mic input gain. If clipping (red zone) appears, audio will distort!

게인 (Gain) Gain 50
