신호 흐름
Cantabile + OBS + Valorant
[Mic] → [Fireface UCX / ASIO] → [Cantabile (VST chain)] → [VB-CABLE]
                                                                  ↓
[Valorant game audio] ———————————————→ [OBS Studio] → YouTube / Twitch
[Backing track / 노래방 mp3] → [Cantabile] ———————————→ (same OBS mix)

ASIO
오디오 인터페이스 — 3 단계
ASIO = 저지연 드라이버. 노래 모니터링 = 10ms 이하. 일반 Windows 드라이버 불가.
Latency = Buffer / Sample Rate
128 / 44,100 = 2.9ms (one-way) — 사람 인지 안 됨 (10ms 이하 = perfect)
Round-trip (UCX II): ~9.16ms — 라이브 보컬 OK
버퍼 선택 가이드
목적 버퍼 지연 상태
라이브 보컬 64~128 1.5~3ms 최적
노래방 + 마이크 (Kenny seed) 128 2.9ms 핵심
OBS 스트리밍 동시 128~256 2.9~5.8ms 안전
후작업 / 믹싱 256~512 5.8~11.6ms 라이브 X
